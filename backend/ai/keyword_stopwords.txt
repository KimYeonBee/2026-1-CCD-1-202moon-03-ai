# 빈칸 키워드 stopword 목록
#
# 이 파일에 등록된 단어는 빈칸으로 출제되지 않음.
# transcript_refiner.py 가 모듈 로드 시 읽어서 frozenset으로 보관.
#
# 형식:
#   - 한 줄에 단어 하나
#   - "#" 으로 시작하는 줄과 빈 줄은 무시
#   - 좌우 공백은 자동 trim
#
# 분류 기준:
#   [universal]      모든 도메인에서 generic — 절대 빈칸 금지
#   [domain-soft]    도메인에 따라 의미가 있을 수 있으나, 보수적으로 제외
#                    (예: 화학 강의의 "물질"은 specific할 수 있지만 너무 일반적)
#
# 새 도메인(역사, 사회, 영어 등) 강의를 추가하면서 학습 효과가 떨어지는
# 단어를 발견하면 [universal] 섹션 또는 [domain-soft] 섹션에 추가하라.
